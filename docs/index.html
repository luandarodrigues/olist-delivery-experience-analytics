Raw → mart analítico

Os CSVs brutos ficam em data/raw/ com Git LFS. O dashboard usa bases derivadas para performance e reprodutibilidade.

Lógica SQL de transformação

WITH item_agg AS (...), payment_agg AS (...), review_agg AS (...) SELECT orders.order_id, customer_state, delivery_days, delay_days, delayed_flag, items_count, sellers_count, freight_value, payment_value, review_score, CASE WHEN review_score <= 2 THEN 1 ELSE 0 END AS low_review_flag FROM orders LEFT JOIN customers LEFT JOIN item_agg LEFT JOIN payment_agg LEFT JOIN review_agg;
Por que importa

O mart transforma tabelas transacionais em uma unidade de negócio: a experiência do pedido.
